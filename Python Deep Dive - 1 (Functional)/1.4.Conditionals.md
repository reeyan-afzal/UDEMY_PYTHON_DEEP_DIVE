### The if Statement (else)

```python
a = 5 

if a < 5:
    print('a < 5')
else:
    print('a >= 5')
```
##### Nested if Statement (elif)

```python
a = 10

if a < 5:
    print('a < 5')
else:
    if a < 10:
        print('5 <= a < 10')
    else:
        print('a > 10')
```

In other languages you have **switch** statement, but in Python there's ```elif```

```python
a = 5 

if a < 5:
    print('a < 5')
elif a < 10:
    print('5 <= a < 10')
elif a < 15:
    print('10 <= a < 15')
elif a < 20:
    print('15 <= a < 20')
else:
    print('a > 20')
```
##### The turnery operator

```python 
'''x if (condition is true) else y'''

a = 25 

if a < 5:
    b = 'a < 5'
else:
    b = 'a >= 5'

print(b)
```

```python
b = 'a < 5' if a < 5 else 'a >= 5'
```
